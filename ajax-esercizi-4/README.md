# 4 Esercizi AJAX dal Screenshot

## Istruzioni
cd ajax-esercizi-4 && python3 -m http.server 8000
Apri http://localhost:8000/ajax-esercizi-4/esercizioN.html

- **esercizio1.html**: Hover su 2 stringhe → 2 zone.
- **esercizio2.html**: Menu 3 opzioni → 3 div specifici.
- **esercizio3.html**: 2 liste 4 stringhe hover → 8 docs in 2 zone.
- **esercizio4.html**: Menu 6 opzioni → 6 docs in 3 span (2 per span).

Tutti file source inclusi. Testa hover/onchange per AJAX!
